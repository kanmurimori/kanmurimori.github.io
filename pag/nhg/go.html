連声

連声れんじょう係部分陽声韵的-n,-m和入声韵-t的音読字与(半)母音起始的音読字和助詞成語和接続時発生的同化過程、其子音与後(半)母音発生連音、只有半母音/y/現在会被保留。後項為/r/的在名称中也有発生、現存一例。

  • /n-(S)V/ → /n-n(y)V/
    ｢銀杏｣ ぎん＋ あん→ ぎんなん
    ｢観音｣*くわん＋*おむ→*くわんのむ　（かんのん）
    ｢云々｣ うん＋ うん→ うんぬん
    ｢反応｣ はん＋ おう→ はんのん
    ｢蘊奥｣ うん＋ おう→ うんのう
    ｢安穏｣ あん＋ おん→ あんのん
    ｢天皇｣ てん＋*わう→ てんなう
    ｢輪廻｣ りん＋*ゑ　→ りんね
    ｢因縁｣ いん＋*ゑん→ いんねん
  • /m-(S)V/ → /m-m(y)V/
    ｢三位｣*さむ＋*ゐ　→*さむみ　　（さんみ）
    ｢陰陽｣*おむ＋*やう→*おむみやう　（おんみょう）
  • /t-(S)V/ → /t-t(y)V/
    ｢雪隠｣ せつ＋ いん→ せつちん　（せっちん）
    ｢屈惑｣ くつ＋ わく→ くつたく　（くったく|屈託）
    ｢出来｣ しゆつ＋ らい→ しゆつたい　（しゅったい）
  • 助詞例
    ｢今日は｣ こんにち＋は→*こんにつた文語
    ｢冥見を｣ みようけん＋を→*みようけんの文語

音便おんびん係在形態素間為発音便宜而音素消失和同化的内連音、可如下四分。其中イウ音便為子音消失、促撥音便為母音消失。ウ撥音便後項為順行同化、促音便前項為逆行同化。音便時最多会発生２個拍的変化。イ促・ウ撥分布相似。

  • ウ音便：/CV[+heigh]c[-voice]/　→　/uc[+voice]/
  • 撥音便：/CV[+heigh]c　　/　→　/Nc[+voice]/
  • イ音便：/Ci　 　c　　/　→　/ic/
  • 促音便：/CV[+heigh]c[-voice]/　→　/Qc/

イ音便おんびん係部分訓読的前高母音拍其子音消失的内連音、若該子音為有声子音則次無声子音将有声化(=同化)。其他前母音音拍也会有此現象、並高母音化(即Ce→i)。内容如下。

  1. /ki(-C)/→/i(-C)/

    埼玉　　/saki-tama/→/sai-tama/
    透垣　　/suki-gaki/→/sui-gai/文語
    妃　　　/kisaki/→/kisai/文語
    濃き　　/kok-i/→/ko-i/　　　濃い
    飽きた　/ak-ita/→/a-ita/　　飽いた
  2. /gi-C/→/i-C[+voice]/
    次手　　/tugi-te/→/tui-de/　序で
    剥ぎて　/hag-ite/→/ha-ide/　剥いで
  3. /si(-C)/→/i(-C)/
    明日　/asi-ta/→/ai-ta/方言・通時
    濃し　　/kos-i/→/ko-i/　　　濃い
    出した　/das-ite/→/da-ite/　出いた文語・方言
  4. /ri(-C)/→/i(-C)/
    ござり　/gozar-i/→/goza-i/　ござい

    奉る　/tatematur-u/→/taimatur-u/文語

連半濁

連半濁れんはんだく係陽声韵尾転写及促音便的音尾在語構成中接合/h/音素時音素変化為/p/的内連音。転写成的音尾｢-N -R -Q｣中除-R之外都有促音化的傾向、其傾向的相対程度-Q＞-N且-Q一定発生 。規則為[Q/N]-h→[Q/N]-p。

動詞的揚音

非複合動詞終止形的揚音有２類平板式・起伏式。平板式揚音為０、大部分起伏式揚音為-２。起伏式中「…エル」形有２類、大部分為動かない形以-２和少量動いた形除-２外還有-３揚音。動いた形下以支持率順列出、動いた形雖有-３揚音但大部分還是傾向於-２。還有部分五段活用的只有或還有-3揚音。一個動詞一般只有一個揚音、少量同時会有0&-2或-2&-3揚音。

０ いる「入る」　いじめる「虐める」　うけたまわる「承る」
-2 いる「居る」　いやがる「嫌がる」　はたらかせる「働かせる」
0|-2 おかす・おかす「侵す」
-2 ← 栄|60整 捕ら 供 狼狽 答 備|30控 踏ま|5訴 携|0|-30考 → -3
※省略｢える｣、上数字為動形与不動形支持率的差。
-2|-3換・蓄・整・支・唱
０|-2|-3出迎・付け加
※這是上面没有収録的語。下面是｢…エル｣以外的-3揚音。
-3覆る・入る/参る・返る/返す・通る/通す・愛す・申す
-2|-3滞る

形容詞的揚音

非複合形容詞終止形的揚音有２類平板式・起伏式。平板式揚音為０、起伏式揚音為-２。
